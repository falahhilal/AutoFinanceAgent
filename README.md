# AutoFinanceAgent

AutoFinanceAgent is an AI-powered agent that analyzes financial transactions to detect subscriptions, estimate usage, and suggest potential cancellations.

## Features
- Detects recurring payments from transaction data  
- Estimates subscription usage based on spending patterns  
- Recommends keep / review / cancel decisions  
- Generates cancellation request drafts  
- Simulates potential savings  

## Tech Stack
- Python  
- LLM (for reasoning)  
- Simulated transaction data
